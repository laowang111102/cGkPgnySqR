# 课程管理系统 java942

## 项目概述

课程管理系统是一款用于辅助教育机构进行课程管理的信息化系统。本项目包含前后端源代码及SQL脚本，提供配套文档，支持一键启动，便于用户快速部署和使用。

## 技术栈

- 前端：HTML5、CSS3、JavaScript、Vue.js
- 后端：Java、Spring Boot、MyBatis
- 数据库：MySQL

## 功能模块

- 用户管理：负责用户的注册、登录、权限分配等功能
- 课程管理：实现对课程信息的添加、修改、删除、查询等功能
- 班级管理：对班级信息进行管理，包括班级的创建、修改、删除等
- 成绩管理：管理学生的课程成绩，包括成绩的录入、修改、查询等
- 其他辅助功能：如通知公告、课程表、教学资源管理等

## 系统角色

- 管理员：负责系统的整体管理，包括用户、课程、班级等信息的维护
- 教师：负责所教授课程的课件上传、成绩录入、公告发布等操作
- 学生：查看课程信息、课程表、成绩以及下载教学资源等

## 运行环境

- 操作系统：Linux或Windows
- Java环境：JDK1.8及以上
- 数据库：MySQL 5.7及以上
- Web服务器：Tomcat 8.5及以上

## 部署步骤

1. 准备Java运行环境，确保MySQL数据库已安装且能正常使用
2. 解压项目压缩包，进入项目目录
3. 执行数据库脚本，初始化数据库
4. 配置项目参数（如数据库连接信息、服务器端口等）
5. 运行项目，访问前端页面进行登录
